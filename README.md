# COJ-CommunityPatch

COJ Community Patch - aiming to fix some bugs, bring back multiplayer. Installer will also check for updates and display changelogs.

It will contain:

-dgVoodoo Accelerator: Fixes duel hands not moving, but sometimes you can't tweak the sensitivity up and down, it stays too high
-Server List Fix Mod
-Multiplayer Old West Map Pack DLC
-Recent Singleplayer Custom Maps (quality maps only)
-Language mods (todo)
-Shorten some Singleplayer Maps (for people on Steam that experience crashes)
-Movies Fix
-Debugscript that skips intro movies, etc.:

Movies Fix, some maps will be downloaded through Google Drive.

For now, this is the list. Whenever I'll have an other idea, I'll update / post it here
